# Feature Baseline: Restaurant POS Android

**Date:** 2026-03-23  
**Status:** Derived directly from the supplied app plan

## Primary Experience

- Cred-style sober pastel interface
- Minimalist table floor view
- Smooth order sheet drawer interactions
- Local-first billing flow
- Simple end-of-day summary

## Table Stakes

- Floor view with clear table occupancy states
- Quick item search by code or name
- Order creation and local persistence per table
- Long-press table switching
- Billing engine with editable tax and service-charge settings
- Offline crash-safe order recovery
- Local PDF generation and print-ready receipts
- Daily summary with revenue, top items, and payment mode split

## Differentiators

- Calm premium aesthetic uncommon in traditional PoS apps
- Subtle animations instead of noisy alerts and popups
- Glassmorphism billing or navigation surfaces
- Sober color cues for veg and non-veg categorization

## Explicit Feature Modules

- Floor view
- Order sheet drawer
- Billing engine
- Quick search
- Daily summary
- Table switching
- Settings for local tax and service-charge rules
