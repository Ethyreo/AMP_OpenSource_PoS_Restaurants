# Docs

This folder is for ongoing project records. The current product direction came from the user on 2026-03-23 and is stored in `product/2026-03-23-app-plan.md`.

## Suggested Use

- `product/` - workflows, visual direction, feature clarifications, and requirement updates
- `architecture/` - runtime choice, storage notes, print strategy, and technical diagrams
- `decisions/` - key tradeoffs and architecture decision records
- `meetings/` - stakeholder notes and action items
- `operations/` - shift, rollout, and support notes
- `hardware/` - Android devices, printers, and payment hardware details
